HTML Introduction

What is HTML?

HTML stands for HyperText Markup Language

  • HyperText: Text with links (non-sequential reading)
  • Markup: Uses tags to mark/define content
  • Language: System of communication and syntax

HTML is the standard markup language used to create web pages. It provides structure and semantic meaning to web content. HTML works with CSS (styling) and JavaScript (interactivity).

Need of HTML?

HTML is essential for several reasons:

  1. STRUCTURE: Provides semantic structure to content
    • Organizes content in a meaningful way
    • Defines different sections (header, nav, main, footer)
  2. BROWSER COMPATIBILITY: Standardized format recognized by all browsers
    • Any browser can interpret and display HTML
  3. SEO OPTIMIZATION: Search engines rely on HTML structure
    • Proper tags help search engines understand content
  4. ACCESSIBILITY: Helps assistive technologies (screen readers)
    • Makes content accessible to people with disabilities
  5. FOUNDATION: Base layer of web development
    • CSS and JavaScript build upon HTML structure

Responsibility of HTML?

HTML has specific responsibilities:

  1. CONTENT MARKUP: Mark up textual and media content
    • Paragraphs, headings, lists, images, videos
  2. SEMANTIC MEANING: Give meaning to content
    • Use <header>, <nav>, <article>, <section> instead of just divs
  3. FORM CREATION: Provide forms for user input
    • Text inputs, checkboxes, radio buttons, submit buttons
  4. LINKING: Connect documents and resources
    • Navigation between pages
    • Linking external stylesheets and scripts
  5. METADATA PROVISION: Include page metadata
    • Title, character encoding, viewport settings

Note: HTML is NOT responsible for:

  • Styling (that's CSS's job)
  • Interactive behavior (that's JavaScript's job)

HTML in React and Other Frameworks

HTML remains fundamental in modern frameworks:

  1. REACT:
    • JSX syntax combines HTML-like syntax with JavaScript
    • React renders HTML to the DOM dynamically
    • Example: <div className="App">Content</div>
  2. ANGULAR:
    • Uses HTML templates with Angular directives
    • Two-way data binding with HTML elements
  3. VUE:
    • Vue templates extend HTML with Vue-specific syntax
    • Components are built around HTML structure
  4. SVELTE:
    • Uses HTML as the template language
    • Compiles components to vanilla JavaScript
  5. NEXT.js / NUXT:
    • Framework-specific components render to HTML
    • Server-side rendering produces HTML

Key Point: All modern frameworks ultimately generate HTML that browsers can render and display.

Elements, Tags, and Attributes

Tags:

The markup syntax used to define elements

Syntax: <tagname>content</tagname>

  • Opening tag: <tagname>
  • Closing tag: </tagname>
  • Self-closing tags: <tagname /> (e.g., <img />, <br />)

Examples of common tags:

<h1>, <p>, <div>, <span>, <a>, <img>, <ul>, <li>

Elements:

Complete unit consisting of opening tag, content, and closing tag

Example: <p>This is a paragraph</p>

  • Opening tag: <p>
  • Content: This is a paragraph
  • Closing tag: </p>
  • Entire structure: The ELEMENT

Attributes:

Extra information about elements

Syntax: <tagname attribute="value">content</tagname>

  • Attributes modify element behavior or provide metadata
  • Always in the opening tag
  • Format: name="value" (value in quotes)

Common attributes:

  • id: Unique identifier for an element
  • class: CSS class for styling
  • href: URL for links
  • src: Source path for images/scripts
  • alt: Alternative text for images
  • width, height: Dimensions
  • disabled: Disables form elements
  • placeholder: Hint text in input fields

Typography (P Tag & Text Formatting)

The <p> Tag in HTML

What is <p> Tag?

  • <p> stands for "Paragraph"
  • It is a block-level element used to define paragraphs of text
  • Each <p> tag creates a new paragraph with automatic spacing before and after
  • Browsers add default margins (typically 16px top and bottom)

Basic Syntax:

<p>This is a paragraph of text.</p>
<p>This is another paragraph.</p>

Key Characteristics of <p> Tag:

  1. BLOCK-LEVEL ELEMENT:
    • Takes full width of parent container
    • Creates line break before and after content
    • Stacks vertically with other block elements
  2. SEMANTIC MEANING:
    • Tells browsers and screen readers this is a paragraph
    • Important for accessibility and SEO
  3. DEFAULT STYLING:
    • Browsers apply default margin: 16px 0
    • Can be overridden with CSS
  4. CONTAINS INLINE ELEMENTS:
    • Can have <strong>, <em>, <a>, <span> inside it
    • Should NOT contain block-level elements like <div>, <h1>

Text Formatting Tags

Text formatting tags are used to format text with special meaning. They add semantic value and visual styling to content.

Emphasis Tags

Tag Purpose Semantic Visual
<strong> Strong importance YES Bold
<em> Emphasis YES Italic
<b> Bold text NO Bold
<i> Italic text NO Italic

Highlighting Tags

  • <mark>: Highlights text (yellow background)
  • <del>: Deleted/strikethrough text
  • <ins>: Inserted/underlined text

Subscript & Superscript

  • <sub>: Subscript text (H2O)
  • <sup>: Superscript text (E=mc2)

Code & Technical Tags

  • <code>: Inline code snippets
  • <kbd>: Keyboard input
  • <var>: Variables
  • <samp>: Sample output

Miscellaneous Tags

  • <small>: Small text
  • <abbr>: Abbreviations
  • <blockquote>: Long quotations
  • <q>: Short inline quotations
  • <cite>: Citations

Best Practices for SEO

  • Use semantic tags (<strong>, <em>) over presentational tags (<b>, <i>)
  • Use proper heading hierarchy (<h1> to <h6>)
  • Emphasize primary keywords naturally
  • Use <mark> for highlighted information
  • Provide full forms for abbreviations with <abbr>

Anchor Tags

What is the Anchor Tag?

Anchor Tag Definition:

The <a> tag stands for "Anchor" and is used to create hyperlinks in HTML. It allows users to navigate to other web pages, files, email addresses, or specific locations within the same page.

Basic Syntax:
<a href="URL">Link Text</a>

Key Characteristics:
• Inline element (doesn't create line breaks)
• Creates clickable hyperlinks
• Essential for navigation and SEO
• Can link to various types of destinations
• Has multiple attributes to modify behavior

Types of Anchor Tags (Links)

  • Absolute Links: Links to external websites with full URL
  • Relative Links: Links to pages within your own website
  • Fragment Links: Links to specific sections within the same page
  • Email Links: Links that open the user's email client
  • Phone Links: Links that trigger phone calls on mobile devices
  • Download Links: Links to downloadable files

Anchor Tag Attributes

Attribute Purpose SEO Impact
href The URL the link points to (REQUIRED) Critical
title Tooltip text on hover, link description High
target Where to open the link Medium
rel Relationship between current page and linked page Very High

Best Practices for SEO

  • Write descriptive anchor text
  • Use internal links strategically
  • Handle external links properly with rel attributes
  • Optimize link title attributes
  • Ensure links are accessible
  • Avoid broken links

Images

What is the Image Tag?

The <img> tag is an HTML element used to embed images in web pages. It is a self-closing tag that does not have a closing tag.

Basic Syntax:
<img src="image.jpg" alt="Description of image">

Essential Attributes

Attribute Purpose Required? SEO Impact
src Path to the image file YES Critical
alt Alternative text describing image YES* Very High
width Image width in pixels NO High
height Image height in pixels NO High
loading Lazy loading strategy NO High

Best Practices for SEO

  • Always include descriptive alt text
  • Use descriptive filenames with keywords
  • Specify width and height to prevent layout shift
  • Use lazy loading for below-the-fold images
  • Optimize image file sizes

Semantic HTML

Semantic HTML uses meaningful tags that describe the content's purpose.

Common Semantic Tags

  • <header> - Page or section header
  • <nav> - Navigation links
  • <main> - Main content
  • <article> - Self-contained content
  • <section> - Thematic grouping
  • <aside> - Sidebar content
  • <footer> - Page or section footer

Media

Audio Element

<audio controls>

Iframe

The <iframe> tag embeds another HTML page within the current page.

Basic Usage

<iframe src="https://www.example.com" width="600" height="400"></iframe>

Security Considerations

  • Use sandbox attribute for security
  • Validate iframe sources
  • Consider same-origin policy

Forms

HTML forms collect user input and send it to a server.

Basic Form Structure

<form action="/submit" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Submit">
</form>

Common Input Types

  • text - Single line text
  • email - Email addresses
  • password - Password fields
  • checkbox - Multiple choice
  • radio - Single choice
  • submit - Form submission

Meta Tags

Meta tags provide metadata about the HTML document.

Common Meta Tags

  • <meta charset="UTF-8"> - Character encoding
  • <meta name="viewport" content="width=device-width, initial-scale=1.0"> - Responsive design
  • <meta name="description" content="Page description"> - SEO description
  • <meta name="keywords" content="keyword1, keyword2"> - Keywords
  • <meta name="author" content="Author Name"> - Author information

Style Tag & CSS

The <style> tag contains CSS rules that style the HTML document.

Internal Stylesheet

<head>
  <style>
    body { background-color: lightblue; }
    h1 { color: navy; }
  </style>
</head>

External Stylesheet

<head>
  <link rel="stylesheet" href="styles.css">
</head>
